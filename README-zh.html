📖 项目介绍

SoloForce（一人公司） 是一款基于 AI 的智能工作流编排系统。

在数字经济时代，个人或小型团队如何高效管理多个 AI 助手完成复杂任务？SoloForce 给出了答案——通过 AI 技术实现一个人管理一个"数字团队"的梦想。

未来的企业将不再依赖传统的"人力资源"，而是转向"数字人力资源"
一个人类 CEO + N 个 AI 数字员工 = 一个高效运转的"一人公司"

📸 项目预览

团队拓扑、员工管理、团队管理、任务中心

✨ 核心特性

1️⃣ 团队拓扑可视化

图形化展示"数字员工"组织和隶属关系，支持拖拽连线建立管理流程。

2️⃣ 自定义数字员工

创建和管理专属的数字员工，定义角色、技能和职责，打造你的虚拟团队。

3️⃣ AI 智能任务分发

输入任务后，AI 自动分析并分发给合适的"数字员工"，模拟真实公司的任务分配流程。

4️⃣ OpenClaw 执行集成

任务执行完成后，提取可执行的技术命令，一键发送到 OpenClaw 执行。

5️⃣ 多团队管理

支持创建多个虚拟团队，每个团队有独立的员工配置和拓扑关系。

🛠️ 技术栈

类别技术
前端React 18 + TypeScript + Vite
可视化React Flow
后端Express + Node.js
AI 引擎OpenAI 兼容 API

🚀 快速开始

安装

git clone https://github.com/yourusername/SoloForce.git
cd SoloForce
npm install

配置

cp .env.example .env
# 编辑 .env 文件，配置你的 API Key

启动

# 终端1: 启动后端服务
npm run server

# 终端2: 启动前端服务
npm run dev

访问 http://localhost:5300

🔧 配置说明

环境变量

PORT=5301

# 大模型 API (必需)
OPENAI_BASE_URL=https://api.siliconflow.cn/v1
OPENAI_MODEL=qwen2.5-7b-instruct
OPENAI_API_KEY=your_api_key_here

# OpenClaw 配置 (可选)
OPENCLAW_URL=http://127.0.0.1:18789
OPENCLAW_TOKEN=your_token

📖 使用指南

步骤操作说明
1️⃣创建团队在"团队管理"页面创建虚拟团队
2️⃣建立拓扑拖拽连接"数字员工"节点
