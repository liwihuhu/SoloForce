## 截图建议

### 团队拓扑页面
- 展示完整的企业组织架构图
- 包含董事长、CEO、员工之间的关系连线
- 突出深海科技风的 UI 设计

### 员工管理页面
- 展示数字员工卡片列表
- 包含搜索/筛选功能
- 展示新增/编辑弹窗效果

### 团队管理页面
- 展示团队卡片列表
- 展示新建/编辑团队弹窗
- 展示团队成员选择界面

### 任务中心页面
- 展示任务统计卡片
- 展示任务历史列表

## 截图尺寸

建议截图尺寸：1600 x 900 像素
